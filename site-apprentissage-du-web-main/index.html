Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Il s'agit de la principale fonctionnalité, principal service d'Internet.Utilisable via un navigateur tel que Firefox, Chrome etc.

Internet

Grossièrement, c'est le réseau de communication entre ordinateurs.

Cloud

Espace de stockage de données et services en ligne.

W3C

Organisme international qui définit les standards techniques liés au web.

Partie 2 - Distinguez les langages du Web

Langages client

Ce sont les langages dit frontend, ils gèrent l'aspect visible du site web. Son affichage.

Navigateurs

Comme le nom l'indique, ils permettent de naviguer sur les différents sites web. De les visiter, les afficher.

Langages serveur

Ce sont les langages dit backend, ils dictent comment se comporte un site, génèrent des réponses aux demandes des clients/utilisateurs.

Frameworks

Ce sont des boîtes à outils, qui permettent d'accélérer le développement grâce à des modèles structurés, standardisés.

Bases de données

C'est un "entrepôt" dans lequel sont stockés les données et dans lequel on peut aller chercher des informations données à l'aide de requêtes.

Site responsive

C'est une façon de coder un site qui permet à ce dernier d'adapter son affichage en fonction du type d'utilisateur : ordinateur, mobile.

Application mobile

Un site ou logiciel spécifiquement développé pour répondre aux besoins d'un mobile.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

C'est là que sont contenus les sites web, et par ce biais qu'ils sont distribués aux clients. Ce sont des machines habituellement stockées dans ce qu'on appelle des datacenter.

Adresse IP

C'est l'adresse d'un ordinateur, chaque ordinateur en possède une. C'est une suite de nombre, on peut comparer ça à une sorte de numéro de téléphone.

Nom d’hôte et DNS

Le nom d'hôte est le nom d'un service, il a été créé pour en faciliter l'usage. Ainsi on peut se rendre sur un site en utilisant son nom, comme google.com. L'annuaire DNS se charge ensuite de faire le lien entre ce nom et l'adresse IP.

Protocoles

C'est ce qu'on pourrait désigner comme étant la/les langues qu'utilisent les ordinateurs pour communiquer entre eux. Ils en existent plusieurs comme TCP qui est un peu la base de tout le reste, ou HTTP qui sert à faire transiter les pages web et que l'on trouve au début de chaque adresses web.

Partie 4 - Identifiez les métiers du web

Développeurs

Il est le concepteur, mais également testeur du site sur lesquels il travaille. Il joue aussi un rôle dans la documentation et maintien de ces derniers.

Product Managers et chefs de projets

Ce sont eux qui s'assurent que le produit répond bien aux besoins des utilisateurs.Ils s'occupent aussi de la vision stratégique et divers objectifs commerciaux, besoins du marché afin de rester compétitifs.

Designers

UX (user experience) et UI (user interface) designers s'occupent de rendre l'expérience utilisateur le plus agréable possible, et de l'attractivité du site.

Testeurs

Il s'occupe d'identifier des divers bugs et autres problèmes de performances.

Techniciens informatique

Il s'occupe du support technique, gestion du parc informatique. Il s'assure que les postes de travail soient opérationnels.

Boîte à outils du développeur

  • IDE comme VS Code :Il s'agit d'un environnement de développement plus confortable et fonctionnel qu'un simple éditeur de texte.
  • Github : Plateforme de stockage en ligne (cloud), elle permet entre autre de partager mais également de travailler efficacement à plusieurs sur un même projet.
  • Librairies / bibliothèques : Comme son nom l'indique il s'agit d'un ensemble de codes pré écrits, compilés qui sont regroupés et qu'on peut aller piocher pour une fonctionnalité spécifique. C'est un gain de temps précieux pour le développement.
  • Frameworks : Véritable boîte à outil simplifiant la vie des développeurs : il en existe beaucoup, par exemple Django pour Python, Symfony pour PHP ou encore J2EE pour Java.
